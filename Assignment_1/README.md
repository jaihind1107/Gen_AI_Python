# Assignment 1: Python Data Structures

A Python assignment covering fundamental data structures: lists, tuples, sets, and dictionaries.

## Prerequisites

- Python 3.6 or higher
- No external dependencies required

## How to Run

```bash
python3 app.py
```

## Tasks Overview

**Task 1: Lists & Tuples**
- Work with product lists
- Index access and append operations
- Convert between tuples and lists

**Task 2: Sets**
- Create sets from lists (removes duplicates)
- Add elements and check membership
- Count unique categories

**Task 3: Dictionaries**
- Store product prices
- Update and delete entries
- Calculate average price
- Find min/max priced products

**Task 4: Combined Operations**
- Create product catalog (list of tuples)
- Build category-to-products mapping
- Find category with most products

## Project Files

- `app.py` - Main script with all 4 tasks
- `README.md` - This documentation

## Learning Outcomes

- Work with lists, tuples, sets, and dictionaries
- Perform CRUD operations on collections
- Combine multiple data structures
- Solve practical problems using Python's built-ins
