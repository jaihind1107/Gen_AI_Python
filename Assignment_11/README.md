
# Assignment 11: Customer Shopping Data Analysis

## Overview
This notebook demonstrates data visualization techniques using pandas and matplotlib on customer shopping data. It includes line plots, scatter plots, bar charts, histograms, and pie charts.

## Prerequisites
```bash
pip install pandas matplotlib
```

## Running the Notebook
1. Ensure `customer_shopping_data.csv` is in the same directory
2. Open the notebook in Jupyter or VS Code
3. Run cells sequentially from top to bottom

## Task Breakdown

**Task 1:** Line plot showing total sales by month  
**Task 2:** Scatter plot comparing sales quantity vs customer age  
**Task 3:** Vertical and horizontal bar charts of monthly sales  
**Task 4:** Multiple bar chart comparing sales across years and months  
**Task 5:** Stacked bar chart for year-over-year sales comparison  
**Task 6:** Histogram showing distribution of sales amounts  
**Task 7:** Pie chart displaying product category distribution  

## Dataset
- **File:** `customer_shopping_data.csv`
- **Key columns:** invoice_date, price, quantity, age, category
